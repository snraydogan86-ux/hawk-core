# Security Policy

## Reporting a vulnerability
If you find a security issue in HAWK, please report it responsibly:

- **Do not** open a public issue for a vulnerability.
- Email the maintainer privately with details and reproduction steps.

We aim to acknowledge reports quickly and to fix confirmed issues promptly.

## HAWK's own safety posture
HAWK Base is aligned to:
- Refuse harmful requests and resist prompt injection.
- Never expose personal or secret data.
- Handle crisis / self-harm situations with a dedicated, deterministic safety layer.

Safety behavior is verified on every model version before promotion.
