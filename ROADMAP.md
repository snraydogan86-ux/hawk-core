# HAWK Roadmap

HAWK is in active development toward global launch. HAWK Base is live and serving real users today.

## Now
- HAWK Base serving 100% of normal conversation on our own GPU.
- Native multilingual (7 languages) across text and voice.
- Evidence-gated model versioning (shadow → canary → production) with automatic rollback.
- Agent OS: memory, tools, orchestration, self-healing.

## Next
- Deeper multilingual and Turkish-naturalness training data (each version).
- Larger own model for the hardest tasks, reducing external dependency further.
- Expanded Workspace (phone → PC) capabilities.
- Continuous self-improvement loop (observe → hypothesize → experiment → evaluate).

## Principle
Own the model. Own the pipeline. Prove it with an open benchmark. Ship safely.
